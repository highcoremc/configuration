# Configuration
Simple configuration abstraction for paper/bungee plugins.

## How to use it?

```xml
<repositories>
  <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
  </repository>
</repositories>
```

```xml
<dependency>
    <groupId>com.github.loperd</groupId>
    <artifactId>configuration</artifactId>
    <version>1.0</version>
</dependency>
```
